# 基础指令集

作为生~~电~~存服，服务器的基础指令将会对你的生存与探索起到一定的帮助，</br>

| 指令       | 用法             | 指令描述            |
|------------|----------------|-----------------|
| `/afk`     | `/afk`         | 挂机开关            |
| `/back`    | `/back`        | 返回上一位置          |
| `/bin`     | `/bin`         | 垃圾桶             |
| `/bs`      | `/bs`          | 主菜单             |
| `/cor`     | `/cor`         | 在线奖励            |
| `/delhome` | `/delhome`     | 删除家             |
| `/sethome` | `/sethome`     | 设置家             |
| `/tpa`     | `/tpa`         | 请求传送到其他玩家        |
| `/tpahere` | `/tpahere`     | 请求传送其他玩家到你这里   |
| `/tpaall`  | `/tpaall`      | 请求传送所有玩家到你这里   |
| `/baltop`  | `/baltop`      | 财富榜             |
| `/pay`     | `/pay <玩家> <金额>` | 付款给其他玩家       |
